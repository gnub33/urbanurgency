An interactive data visualization platform exploring how transportation infrastructure influences environmental, economic, and social outcomes in the United States.

## Overview

Urban Urgency is a React-based web application that transforms public transportation and environmental datasets into interactive visualizations. The platform enables users to explore relationships between transportation infrastructure, land use, emissions, and mobility through an intuitive, data-driven interface.

The project demonstrates modern frontend development, data visualization techniques, and the ability to communicate complex datasets through interactive web experiences.
